# Hardware — LoRa Companion (aspirational, v2+)

This directory is a placeholder for notes on a possible physical companion
device (a small LoRa-connected object the player's AI could theoretically
"live in" or reach through). This is explicitly a **hardware side-project**,
not part of the mobile app roadmap — see `docs/brief.md` §2 (OUT of v1) and
`docs/future.md`.

Nothing here is built or scheduled. Do not wire the app to expect hardware
that doesn't exist. If you're picking this up as a side-project, start a dated
notes file in this directory rather than editing this README in place.
